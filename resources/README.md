# resources/

This directory contains all data files and vendored source trees that are
either too large or too platform-specific to be installed from PyPI.

---

## Subdirectories

### `LeptonInjector/`
Vendored source of the [icecube/LeptonInjector](https://github.com/icecube/LeptonInjector)
`with_earth_py` branch (commit `d203189b`, Feb 2023, LGPL-3.0).

Built and installed by `scripts/install_leptoninjector_legacy.sh`.  The Docker
image also builds from this directory.  Do **not** clone from GitHub at install
time — use this vendored copy so that the four GCC 13 / SuiteSparse 7.x
compatibility patches are always present.

**To update:** replace the directory contents with the new upstream commit,
re-apply any patches that are still needed, and test with
`source scripts/activate.sh .prometheus_env && python -c "import LeptonInjector"`.

---

### `fennel/`
Vendored source of [fennel-seed](https://github.com/MeighenBergerS/fennel)
v2.0.0 (commit `988bf2f`, MIT license).

fennel-seed is not published on PyPI; this copy removes the need for a GitHub
clone at install time.  Installed by `scripts/fixes.sh` and by the Docker image.

`notebooks/` and `seed/` from the upstream repository are excluded as they are
not needed at runtime.

**To update:** copy the new release into this directory, excluding `notebooks/`
and `seed/`, and update the version note here.

---

### `PPC_executables/`
Source code of the Photon Propagation Code (PPC).

| Sub-directory | Contents |
|---|---|
| `PPC/` | CPU-only version.  Built with `make cpu`. |
| `PPC_CUDA/` | CUDA GPU version.  Built with `make gpu arch=<SM>`. |

Both are built by `scripts/install_ppc.sh` (CPU only) or by the GPU Docker
image.  See `PPC_executables/README.md` for build details.

---

### `PPC_tables/`
Pre-computed ice/water photon-propagation lookup tables consumed by PPC at
runtime.

| Sub-directory | Medium |
|---|---|
| `south_pole/` | South Pole glacial ice (IceCube site) |
| `spx/` | SPX ice model variant |

The path to the table directory is read from `config["photon propagator"]["ppc"]["ppctables"]`.

---

### `PROPOSAL_tables/`
Cached cross-section / interpolation tables generated by the PROPOSAL lepton
propagator.  **Generated automatically** the first time a simulation is run
(~1–2 min); subsequent runs are instant.

This directory is listed in `.gitignore` — do not commit its contents.

---

### `cross_section_splines/`
FITS-format neutrino cross-section splines used by LeptonInjector for
volume and ranged injection.

| File | Description |
|---|---|
| `dsdxdy_nu_CC_iso.fits` | ν CC differential cross section |
| `dsdxdy_nu_NC_iso.fits` | ν NC differential cross section |
| `dsdxdy_nubar_CC_iso.fits` | ν̄ CC differential cross section |
| `dsdxdy_nubar_NC_iso.fits` | ν̄ NC differential cross section |
| `sigma_nu_CC_iso.fits` | ν CC total cross section |
| `sigma_nu_NC_iso.fits` | ν NC total cross section |
| `sigma_nubar_CC_iso.fits` | ν̄ CC total cross section |
| `sigma_nubar_NC_iso.fits` | ν̄ NC total cross section |

---

### `earthparams/`
Earth density and material profiles consumed by LeptonInjector's
`EarthModelService`.  The path is exported as `EARTH_PARAMS` by the install
scripts and the Docker image.

---

### `geofiles/`
Detector geometry files (`.geo` format) describing the positions and properties
of optical modules for each detector.

| File | Detector |
|---|---|
| `demo_water.geo` | Small demo water detector (used by `examples/01_basic_water.py`) |
| `demo_ice.geo` | Small demo ice detector (used by `examples/02_basic_ice.py`) |
| `icecube.geo` | IceCube (86 strings) |
| `icecube_gen2.geo` | IceCube-Gen2 |
| `deepcore.geo` | DeepCore sub-array |
| `upgrade.geo` | IceCube Upgrade |
| `arca.geo` | KM3NeT/ARCA |
| `orca.geo` | KM3NeT/ORCA |
| `gvd.geo` | Baikal-GVD |
| `pone_triangle.geo` | P-ONE triangular geometry |
| `trident.geo` | TRIDENT |

Geo files are loaded with `prometheus.detector.detector_from_geo(path)`.

---

### `olympus_resources/`
Pre-trained model weights for the Olympus normalizing-flow photon propagator.

| File | Contents |
|---|---|
| `photon_arrival_time_nflow_params.pickle` | Normalizing-flow network weights for photon arrival-time prediction |
| `photon_arrival_time_counts_params.pickle` | Network weights for photon count prediction |
| `pone_config.json` | P-ONE-specific Olympus configuration |

---

## Vendoring policy

A dependency is vendored here when:
1. It is **not available on PyPI** (fennel-seed), or
2. It requires **source-level patches** that are not yet merged upstream
   (LeptonInjector GCC 13 / SuiteSparse 7.x fixes), or
3. It must be present **without internet access** during install (both of the above).

Do not vendor dependencies that can be satisfied by a plain `pip install`.
